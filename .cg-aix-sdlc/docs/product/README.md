## Quick Reference

### What is LangBuilder?

LangBuilder is a visual AI workflow builder that enables teams to create, test, and deploy AI-powered applications using a drag-and-drop interface. Built on LangChain, it provides:

- **24+ LLM providers** - OpenAI, Anthropic, Google, and more
- **19+ vector stores** - Pinecone, ChromaDB, Qdrant, and more
- **30+ integrations** - CRM, search, documents, and enterprise tools
- **OpenAI-compatible API** - Drop-in replacement for existing apps
- **Self-hosted deployment** - Full data sovereignty

### Key Value Propositions

1. **Build Faster** - Visual interface eliminates boilerplate code
2. **Stay Flexible** - No vendor lock-in with 24+ LLM providers
3. **Maintain Control** - Self-hosted deployment for data sovereignty
4. **Scale Confidently** - Production-ready with comprehensive API

### Target Users

- AI/ML Engineers
- Backend Developers
- Platform Teams
- Technical Product Managers
